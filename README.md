# TrailStories

Persoonlijk outdoor storytelling platform — wandelingen vastleggen met GPS-data, foto's en een persoonlijk verhaal.

TrailStories combineert wandelroutes met GPS-tracks, fotografie en praktische info tot een visueel "trail story" per wandeling. Het project is een static, frontend-only web app (vanilla HTML/CSS/JS) zonder backend in de MVP-fase.

## Databronnen

| Bron | Gebruik |
|------|---------|
| [GPX Viewer](https://gpxviewer.com) | Registratie van de wandeling onderweg (tracking) |
| [AllTrails](https://www.alltrails.com) | Verkenning van routes, download van GPX-kaarten |
| [OpenStreetMap](https://www.openstreetmap.org) | Kaartlaag / referentie voor route-tracé |

## Tech stack

HTML5 · CSS3 · Vanilla JavaScript · Leaflet.js · OpenStreetMap tiles · JSON als single source of truth
